Frontend Mentor - Clipboard Landing Page Solution

This is my solution to the Clipboard Landing Page challenge on Frontend Mentor. The project helped me practice responsive web design, Flexbox, CSS Grid, and creating layouts that work across different screen sizes.


### Challenge

Users should be able to:

* View the optimal layout for the site depending on their device's screen size.
* See hover states for interactive elements.

### What I Learned

While building this project, I improved my understanding of:

* Creating responsive layouts using Flexbox and CSS Grid.
* Structuring sections with semantic HTML.
* Using media queries for smaller screen sizes.
* Managing spacing, alignment, and typography consistently.
* Building reusable button styles and responsive components.

Example of responsive layout implementation:

```css
@media (max-width: 700px) {
    .feature-list {
        flex-direction: column;
        justify-content: center;
    }

    .items {
        text-align: center;
    }
}
```
